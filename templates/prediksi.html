1️⃣ Apa Algoritma Yang Digunakan?
Multiple Linear Regression dengan Polynomial Features (Degree 2) menggabungkan dua konsep:
❌ Simple Linear Regression
  • Persamaan: y = β₀ + β₁x
  • Hanya 1 feature
  • Prediksi garis lurus
  • Kurang akurat data kompleks
✅ Multiple Linear Regression (Kami)
  • Persamaan: y = β₀ + β₁x + β₂x²
  • 2 features (tahun + tahun²)
  • Prediksi mengikuti kurva
  • Lebih akurat data dinamis
2️⃣ Rumus Matematika
Persamaan Model

📝 Persamaan Prediksi

$$\hat{y} = \beta_0 + \beta_1 \cdot x + \beta_2 \cdot x^2$$
Dimana:
  • ŷ = Nilai prediksi (populasi)
  • x = Tahun
  • β₀ = Intercept (nilai dasar)
  • β₁ = Koefisien linear (kecepatan)
  • β₂ = Koefisien kuadrat (akselerasi)
Feature Engineering

🔄 Transformasi Polynomial

Dari 1 feature menjadi 2 features:
$$X_{original} = [x_1, x_2, ..., x_n]$$
$$X_{poly} = [1, x_1, x_1^2, x_2, x_2^2, ..., x_n, x_n^2]$$
3️⃣ Proses Perhitungan Step-by-Step

📊 Langkah-Langkah

  1. Persiapan Data: Ambil data historis daerah (tahun 2019-2025)
  2. Transformasi Feature: Ubah tahun menjadi polynomial: x dan x²
  3. Training Model: Hitung β₀, β₁, β₂ dengan Least Squares
  4. Validasi Model: Hitung R² Score dan RMSE
  5. Prediksi Masa Depan: Gunakan model untuk tahun 2026-2030
  6. Output: Tampilkan prediksi dengan metrics akurasi

4️⃣ Metrik Akurasi Model
R² Score (Coefficient of Determination)

📊 Rumus

$$R^2 = 1 - \frac{SS_{res}}{SS_{tot}} = 1 - \frac{\sum (y_i - \hat{y}_i)^2}{\sum (y_i - \bar{y})^2}$$
Interpretasi:
  • R² = 1.0 → Model sempurna (100% akurat)
  • R² = 0.5 → Model menjelaskan 50% variance
  • R² = 0.0 → Model tidak menjelaskan apa-apa
RMSE (Root Mean Squared Error)

📏 Rumus

$$RMSE = \sqrt{\frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2}$$
Rata-rata error dalam unit ribu jiwa. Semakin kecil, semakin baik.

6️⃣ Random Forest Regressor
🌲 Apa Itu Random Forest?
Random Forest adalah algoritma ensemble yang membangun banyak decision trees secara acak dan menggabungkan prediksi mereka untuk hasil yang lebih akurat dan stabil.
🔄 Cara Kerja Random Forest
  1. Bootstrap Sampling: Ambil N sample random dari data original dengan replacement (bagging)
  2. Build Decision Trees: Untuk setiap sample, buat decision tree dengan menggunakan random features
  3. Random Feature Selection: Pada setiap split, hanya gunakan subset random dari features
  4. Multiple Trees: Repeat sampai N trees terbentuk (kami: 100 trees)
  5. Aggregation: Untuk regresi, rata-ratakan prediksi dari semua trees
  6. Final Prediction: Hasil rata-rata dari 100 trees adalah prediksi akhir
📐 Formula Agregasi

Prediksi Random Forest

$$\hat{y}_{RF} = \frac{1}{N} \sum_{i=1}^{N} T_i(x)$$
Dimana:
• ŷ_RF = Prediksi Random Forest
• N = Jumlah trees (100)
• T_i(x) = Prediksi dari tree ke-i
• x = Input tahun
🛠️ Parameter Random Forest Kami
  • 🌳 n_estimators = 100 → Jumlah trees yang dibangun
  • 📊 max_depth = 5 → Kedalaman maksimal setiap tree (mencegah overfitting)
  • 🔗 min_samples_split = 2 → Minimum sample untuk split node
  • 🎲 random_state = 42 → Seed untuk reproducibility
  • ⚡ n_jobs = -1 → Parallel processing (gunakan semua CPU cores)
✅ Keunggulan Random Forest
Random Forest vs Polynomial
  • ✅ Tangkap non-linear patterns lebih baik
  • ✅ Robust terhadap outliers/anomali
  • ✅ Tidak perlu feature scaling
  • ✅ Built-in feature importance
  • ✅ Parallelisasi cepat
  • ✅ Lebih flexible untuk berbagai data patterns
⚠️ Pertimbangan
  • Lebih black-box (sulit diinterpretasi)
  • Butuh lebih banyak komputasi
  • Perlu tuning hyperparameter
  • Bisa overfitting jika tidak careful
  • Training time lebih lama
  • Prediksi tidak linear seperti polynomial
