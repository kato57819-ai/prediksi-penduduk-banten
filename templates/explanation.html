1️⃣ Apa Algoritma Yang Digunakan?
Multiple Linear Regression dengan Polynomial Features (Degree 2) menggabungkan dua konsep:
❌ Simple Linear Regression
  • Persamaan: y = β₀ + β₁x
  • Hanya 1 feature
  • Prediksi garis lurus
  • Kurang akurat data kompleks
✅ Multiple Linear Regression (Kami)
  • Persamaan: y = β₀ + β₁x + β₂x²
  • 2 features (tahun + tahun²)
  • Prediksi mengikuti kurva
  • Lebih akurat data dinamis
2️⃣ Rumus Matematika
Persamaan Model

📝 Persamaan Prediksi

$$\hat{y} = \beta_0 + \beta_1 \cdot x + \beta_2 \cdot x^2$$
Dimana:
  • ŷ = Nilai prediksi (populasi)
  • x = Tahun
  • β₀ = Intercept (nilai dasar)
  • β₁ = Koefisien linear (kecepatan)
  • β₂ = Koefisien kuadrat (akselerasi)
Feature Engineering

🔄 Transformasi Polynomial

Dari 1 feature menjadi 2 features:
$$X_{original} = [x_1, x_2, ..., x_n]$$
$$X_{poly} = [1, x_1, x_1^2, x_2, x_2^2, ..., x_n, x_n^2]$$
3️⃣ Proses Perhitungan Step-by-Step

📊 Langkah-Langkah

  1. Persiapan Data: Ambil data historis daerah (tahun 2019-2025)
  2. Transformasi Feature: Ubah tahun menjadi polynomial: x dan x²
  3. Training Model: Hitung β₀, β₁, β₂ dengan Least Squares
  4. Validasi Model: Hitung R² Score dan RMSE
  5. Prediksi Masa Depan: Gunakan model untuk tahun 2026-2030
  6. Output: Tampilkan prediksi dengan metrics akurasi

5️⃣ Metrik Akurasi Model
R² Score (Coefficient of Determination)

📊 Rumus

$$R^2 = 1 - \frac{SS_{res}}{SS_{tot}} = 1 - \frac{\sum (y_i - \hat{y}_i)^2}{\sum (y_i - \bar{y})^2}$$
Interpretasi:
  • R² = 1.0 → Model sempurna (100% akurat)
  • R² = 0.5 → Model menjelaskan 50% variance
  • R² = 0.0 → Model tidak menjelaskan apa-apa
RMSE (Root Mean Squared Error)

📏 Rumus

$$RMSE = \sqrt{\frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2}$$
Rata-rata error dalam unit ribu jiwa. Semakin kecil, semakin baik.

6️⃣ Keunggulan & Limitasi
✅ Keunggulan
  • Tangkap trend non-linear
  • Lebih akurat dari linear
  • Cepat dalam perhitungan
  • Mudah diinterpretasi
  • Cocok untuk time series
  • Metrics akurasi transparan
⚠️ Limitasi
  • Asumsi hubungan polynomial
  • Tidak tangani anomali
  • Data training terbatas (5 tahun)
  • Prediksi lama kurang reliable
  • Abaikan faktor eksternal
  • Butuh validasi berkala

🔬 Metode: Linear Regression

Penjelasan lengkap tentang algoritma yang digunakan sistem untuk menghitung prediksi populasi Banten

⚙️ Bagaimana Cara Kerjanya?

Linear Regression adalah algoritma machine learning yang paling sederhana dan populer. Cara kerjanya adalah dengan menganalisis data historis, kemudian menarik garis lurus yang paling sesuai dengan semua data tersebut.

Langkah-langkah:

  1. Kumpulkan data historis → Kami mengambil data populasi Banten 6 tahun terakhir dari BPS (2020-2025)
  2. Analisis tren → Algoritma menemukan hubungan antara tahun (X) dan jumlah penduduk (Y)
  3. Hitung parameter → Sistem menghitung nilai m (slope/kemiringan) dan b (intercept/perpotongan)
  4. Buat prediksi → Garis tersebut dilanjutkan ke tahun-tahun mendatang
📐 Rumus Matematika
y = mx + b

Keterangan:

  • y = Jumlah penduduk (yang diprediksi)
  • x = Tahun
  • m = Slope (kemiringan/tingkat pertumbuhan tahunan)
  • b = Intercept (nilai awal/konstanta)
🎯 Interpretasi Hasil

📈 Slope Positif (m > 0)

Populasi terus bertambah setiap tahunnya. Ini adalah kasus Banten - pertumbuhan populasi positif.

📉 Slope Negatif (m < 0)

Populasi berkurang setiap tahunnya. Contoh: negara dengan tingkat kelahiran rendah.

➡️ Slope Nol (m = 0)

Populasi tetap stabil dari tahun ke tahun (sangat jarang terjadi).

✓ Asumsi Model

Model ini mengasumsikan beberapa hal yang mungkin atau mungkin tidak terjadi:

  • ✓ Tren konsisten - Pertumbuhan populasi tetap mengikuti pola yang sama
  • ✓ Tidak ada perubahan drastis - Tidak ada bencana alam, epidemi, atau migrasi massal
  • ✓ Data akurat - Semua data historis sudah benar dan lengkap
  • ✓ Hubungan linear - Tidak ada kurva atau perubahan pola yang kompleks
  • ✓ Stabilitas parameter - Tingkat kelahiran, kematian, dan migrasi relatif stabil
⚠️ Keterbatasan & Akurasi

Penting untuk diketahui:

  • • Prediksi ini adalah estimasi berdasarkan pola historis, bukan kepastian
  • • Akurasi berkurang untuk tahun-tahun yang jauh ke depan (2028-2030 lebih tidak pasti dari 2025-2026)
  • • Model ini sangat sederhana - tidak mempertimbangkan faktor-faktor kompleks seperti:
    • 🔴 Program Keluarga Berencana (KB)
    • 🔴 Krisis ekonomi atau resesi
    • 🔴 Pandemi atau bencana alam
    • 🔴 Perubahan kebijakan migrasi
    • 🔴 Peningkatan harapan hidup
  • • Untuk prediksi yang lebih akurat, gunakan model yang lebih canggih (Polynomial Regression, ARIMA, Neural Networks, dll)
📊 Contoh Perhitungan dari Data Banten

Mari lihat bagaimana Linear Regression menghitung prediksi dari data BPS Banten:

📈 Data Historis Banten (2020-2025):

Tahun Total Penduduk Laki-laki Perempuan
2020 11,904,562 6,070,271 5,834,291
2021 12,038,952 6,134,423 5,904,529
2022 12,173,342 6,198,575 5,974,767
2023 12,307,732 6,262,727 6,045,005
2024 12,431,400 6,320,900 6,110,500
2025 12,537,400 6,369,700 6,167,700

🔢 Langkah Perhitungan:

  1. Hitung pertumbuhan rata-rata per tahun:
    Slope (m) = (Populasi_2025 - Populasi_2020) / (2025 - 2020)
    m = (12,537,400 - 11,904,562) / 5
    m = 632,838 / 5
    m ≈ 126,568 jiwa per tahun
  2. Hitung nilai awal (intercept):
    b = Populasi_2020 - (m × 2020)
    b = 11,904,562 - (126,568 × 2020)
    b = 11,904,562 - 255,663,760
    b ≈ -243,759,198
  3. Prediksi tahun 2026:
    y = m × x + b
    Populasi_2026 = 126,568 × 2026 + (-243,759,198)
    Populasi_2026 ≈ 12,663,968 jiwa
  4. Prediksi tahun 2030:
    Populasi_2030 = 126,568 × 2030 + (-243,759,198)
    Populasi_2030 ≈ 13,043,242 jiwa

Kesimpulan: Banten mengalami pertumbuhan konsisten sekitar 126,568 jiwa per tahun berdasarkan data 2020-2025. Jika tren ini berlanjut, populasi Banten diperkirakan mencapai 12,66 juta di 2026 dan 13,04 juta di 2030.
